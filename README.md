## Instalaciones 

1. Tener Nest CLI instalado
```
npm i -g @nestjs/cli
```

2. Instalar dependencias
```
$ npm install
```

3. Levantar la base de datos
```
docker-compose up -d
```
## Correr la aplicacion

```
$ npm run start:dev
```

## Stack usado
* MongoDB
* Nest
* Mongoose
